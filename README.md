# TP-Statistical-Learning-Model
This Model Simulates an experiment from Saffran et al. (1996) and Aslin et al. (1998) using the same stimuli.
It works as an upper boundary to what can be learned only from statistical analysis of the bigram frequencies or
transitional probability when it comes to word separation.
The model is part of the essay "Statistical Learning in Pre-lexical Infants: How Far Does Transitional Probability Get Us?" for the seminar "Language and Cognition". 

## Run the Experiment
To run the experiment and get results plotted using matlab just run the "RunExperiment.py" file.
The number of trials can be changed in the main-method.
The experiment number indicates which experiment to run:
- expnr. = 0 -> Saffran et al. (1996)
- expnr. = 1 -> Aslin et al. (1998)
